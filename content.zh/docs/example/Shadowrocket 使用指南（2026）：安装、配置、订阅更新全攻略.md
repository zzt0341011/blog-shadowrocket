可以把 Shadowrocket（中文俗称“小火箭”）理解为：苹果平台上最知名的“规则分流代理客户端”之一。

它本身不是 VPN 服务商，而是一个 代理协议客户端，支持 Shadowsocks、ShadowsocksR、VLESS、Trojan、VMess、Hysteria 等多种协议，并提供非常强大的 规则分流、订阅导入、广告屏蔽 和 流量统计 功能。很多中国用户会把它当作 iPhone/iPad 上的“代理管理中心”来使用。



| 类型      | 代表                           |
| ------- | ---------------------------- |
| 普通 VPN  | 一键连接型                        |
| 高级代理客户端 | Shadowrocket                 |
| 桌面同类    | Clash Verge Rev、v2rayN、Surge |

### 为什么中国 App Store 找不到？

### 核心原因：大陆区下架

自 2017 年起

最关键的原因是：苹果在 2017 年按照中国大陆监管要求，下架了大量 VPN/代理类应用，Shadowrocket 就在其中。

当时被下架的还包括 Surge、Potatso 2 等工具。苹果官方的说法是：这些应用不符合中国大陆的新规定，因此只能从 中国大陆区 App Store 移除，但在其他国家和地区仍可下载。





### 现在还能下载吗？

### 可以，但取决于 Apple ID 地区

重点

| Apple ID 地区 | 是否能下载 |
| ----------- | ----- |
| 中国大陆        | 不能    |
| 香港          | 可以    |
| 美国          | 可以    |
| 新加坡         | 可以    |
| 日本          | 可以    |

例如香港区和美国区 App Store 目前仍然能找到 Shadowrocket。



### 为什么它在中国用户中这么火？

Shadowrocket（俗称“小火箭”）之所以在中国用户中广受欢迎，主要得益于其强大的功能、稳定的性能以及成熟的使用生态。作为 iPhone 和 iPad 平台知名的 iOS 代理客户端，Shadowrocket 支持 VLESS、Trojan、Shadowsocks、VMess、Hysteria 等多种主流代理协议，能够兼容绝大多数节点和订阅服务。用户只需导入订阅链接，即可快速完成配置，无需复杂操作，使用门槛较低。同时，它提供灵活的规则分流功能，可根据域名、IP 地址、地理位置等条件智能选择连接方式，实现国内网站直连、海外网站代理，在保证访问速度的同时提升网络体验。此外，Shadowrocket 还集成了广告拦截、DNS 管理、脚本扩展、MITM 配置、流量统计等丰富功能，满足从普通用户到高级用户的多样化需求。经过多年发展，Shadowrocket 已形成完善的教程、规则集和订阅生态，许多代理服务商都提供针对其优化的配置，因此成为众多 iOS 用户长期使用和推荐的代理工具，在同类应用中保持着较高的知名度和影响力。



### 它和普通 VPN 的区别（最容易混淆）

| 项目      | Shadowrocket | 普通 VPN |
| ------- | ------------ | ------ |
| 本质      | 代理客户端        | VPN 服务 |
| 是否自带服务器 | 否            | 是      |
| 需要节点/订阅 | 需要           | 通常不需要  |
| 规则分流    | 非常强          | 一般     |
| 适合高级用户  | 是            | 否      |
| 适合一键使用  | 一般           | 是      |
